### What Counts as a "Pattern"?

A pattern is the underlying psychological mechanism or positioning strategy, not the specific angle name:

- **"Foundation/calm first"** = positioning that regulation must come before action
- **"Permission giving"** = framing that removes parental guilt
- **"Science made simple"** = translating research into parent-friendly language
- **"Insider knowledge"** = revealing what experts know but parents don't

### When to Add a Pattern

Add to Pattern Tracking when:
- The same underlying mechanism is used in 2+ angles (even if angle names differ)
- A core differentiator is used (mark immediately for tracking)

Do NOT add every angle; only recurring themes that risk becoming stale.

### Pattern Reuse Justification

If reusing a pattern across articles:
1. Different pillar = acceptable (e.g., "calm first" in Sleep pillar vs ADHD pillar)
2. Same pillar = requires strong justification (avoid if possible)
3. Document reasoning in Notes column
